# Health Insurance Analytics Dashboard

## Project Overview

This project demonstrates a comprehensive Power BI solution for analyzing health insurance claims, operational performance, fraud detection, customer behavior, and hospital performance.

---

## Business Objective

To enable insurance companies to monitor KPIs, reduce fraudulent claims, improve operational efficiency, and gain actionable business insights through interactive dashboards.

---

## Dashboard Modules

- Executive Dashboard
- Claims Dashboard
- Fraud Detection
- Customer 360
- Hospital 360
- Agent Performance
- Trend Analysis
- Financial Overview

---

## Tools Used

- Power BI Desktop
- SQL
- Power Query
- DAX

---

## Skills Demonstrated

- Dashboard Development
- Data Modeling
- KPI Reporting
- Interactive Visualizations
- Drill-through
- Bookmarks
- Dynamic Filtering
- Business Intelligence
