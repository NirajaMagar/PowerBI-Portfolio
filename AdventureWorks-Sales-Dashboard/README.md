# Adventure Works Sales Dashboard

## Project Overview

This Power BI dashboard provides an executive view of sales performance using the Adventure Works dataset.

---

## Business Objective

The objective of this dashboard is to help business stakeholders monitor sales performance, identify top-performing products, analyze customer purchasing trends, and improve decision-making through interactive visualizations.

---

## Tools Used

- Power BI Desktop
- SQL
- Power Query
- DAX
- Data Modeling

---

## Dashboard Features

- Executive KPI Cards
- Sales Trends
- Product Analysis
- Customer Analysis
- Geographic Sales Analysis
- Dynamic Filters
- Drill-through Pages
- Interactive Charts

---

## Key KPIs

- Total Sales
- Total Orders
- Total Customers
- Profit
- Sales Growth
- Average Order Value
